# Projet Flutter : Reproduction d'une maquette Dribbble

## Objectif

Choisir une interface utilisateur (UI) sur Dribbble et la reproduire le plus fidèlement possible en Flutter.

## Consignes

1. Rendez-vous sur https://dribbble.com.
2. Choisissez une maquette de votre choix.
3. Reproduisez l'interface en Flutter en respectant au maximum :

   * Les couleurs ;
   * Les polices ;
   * Les espacements ;
   * Les dimensions ;
   * Les icônes ;
   * L'organisation générale des composants.
4. L'application doit être fonctionnelle et responsive.
5. Les widgets Flutter doivent être organisés de manière propre et réutilisable.

## Livrables

Le dépôt GitHub devra contenir :

* Le code source Flutter ;
* Une capture d'écran de la maquette originale ;
* Une ou plusieurs captures d'écran de votre réalisation ;
* Un fichier `README.md` contenant :

  * Le lien de la maquette Dribbble choisie ;
  * Une courte description du projet ;
  * Les éventuelles difficultés rencontrées.

## Critères d'évaluation

| Critère                | Points |
| ---------------------- | ------ |
